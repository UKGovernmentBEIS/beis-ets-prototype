Set up two-factor authentication

Two-step verification adds additional protection to your account. When you sign in to the Emissions Trading Registry you'll need to enter both your password and a verification code. This verification code will be generated on your mobile device with an authenticator app.

When you access your account or do any kind of transaction on the system, you will always need your password and either one of the following:

  • A verification code (from the authenticator app)
  • A backup code
